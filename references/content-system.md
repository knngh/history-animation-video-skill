# Content System

Use this reference when designing the account, content pillars, scripts, storyboards, or calendars.

## Positioning Formula

Format:

`给【人群】看的【历史人物】动画短视频，用【视觉风格】讲清【人物关键选择】。`

Examples:

- 给年轻上班族看的中国历史人物动画，用电影感讲清权力、选择和代价。
- 给海外华人看的双语中国人物短片，用国风动画讲清一个人物为什么被历史记住。
- 给中学生看的历史人物故事，用漫画动画把课本人物讲得像短剧。

## Niche Choices

Choose one narrow wedge first:

- Dynasty wedge: Qin-Han, Tang-Song, Ming-Qing.
- Character wedge: emperors, generals, reformers, women in history, poets, scientists.
- Conflict wedge: betrayal, exile, failed reform, final battle, last letter, one decision.
- Audience wedge: exam-friendly history, workplace leadership, women-focused history, overseas bilingual history.
- Visual wedge: cinematic realism, ink animation, comic panels, paper-cut, documentary collage.

Avoid starting with "all history". A repeatable wedge makes production and audience memory easier.

## Topic Matrix

Use this matrix to generate topics:

`人物 + 关键选择 + 代价 + 反常识 + 可视化场景`

Good topic examples:

- 秦始皇统一后，为什么反而更不安？
- 霍去病打赢的关键，为什么不是勇猛？
- 张居正改革成功，为什么身后失败？
- 李清照南渡后，失去的不只是爱情。
- 王阳明最难的战场，不在战场。

Reject weak topics:

- "某某的一生" - too broad.
- "三分钟讲完某朝代" - too encyclopedic.
- "十大历史人物" - hard to build character attachment.

## Viral Story Engine

Build every short around this engine:

`Open loop -> pressure -> irreversible choice -> price -> debate`

Examples:

- Open loop: "历史记住他赢了，却很少讲他赢完之后失去了什么。"
- Pressure: "退一步，他会被旧势力吞掉；进一步，他会得罪所有人。"
- Irreversible choice: "他把名字写上去的那一刻，就没有回头路了。"
- Price: "胜利没有让他安全，只让更多人开始害怕他。"
- Debate: "这是果断，还是赌徒式冒险？"

If the story has no irreversible choice, rewrite the topic before writing the script.

## Script Structure

Use 6 beats:

1. Hook: one question, contradiction, or dangerous moment.
2. Setup: one sentence explaining who/where/when.
3. Pressure: what makes the choice hard.
4. Decision: what the character does.
5. Consequence: what changes because of it.
6. Aftertaste: what modern viewers should remember.

Keep one video to one core question. If a paragraph does not serve that question, cut it.

## Retention Rules

- Put the strongest contradiction in the first sentence.
- Use one concrete object before second 8: map, letter, sword, seal, candle, gate, rain.
- Make every 2-4 seconds visually different.
- Cut background context to one sentence.
- Do not explain the entire life. Show one decision and its cost.
- End with a debate prompt that splits reasonable viewers into two sides.

## Shot Structure

For 45-90 seconds, use 6-10 shots:

1. Title hook or black-screen question.
2. Character close-up.
3. Symbolic object: map, sword, letter, seal, candle, gate, rain.
4. Pressure scene: court, camp, prison, road, study.
5. Decision close-up.
6. Consequence wide shot.
7. Return to face.
8. End card with question.

Each shot should have one action only. Complex motion increases AI video artifacts.

## Distribution Loop

For every finished video, create:

- 5 titles: curiosity, conflict, emotion, reversal, search-friendly.
- 3 cover texts: danger, choice, cost.
- 3 first-line variants.
- 2 end cards: debate and part two.
- 1 comment seed: a neutral but debatable first comment.

## 30-Day Launch System

- Week 1: positioning, visual style, 20 topics, 10 scripts.
- Week 2: character sheets and first 5 videos.
- Week 3: publish and observe, keep style stable.
- Week 4: double down on best hook pattern, stop weak columns.

Publish enough to learn. Do not spend the first month perfecting tools.

Daily publishing test:

- Day 1-10: test hook type.
- Day 11-20: test character niche.
- Day 21-30: test platform packaging.

Track first-frame hold, 3-second retention, completion rate, rewatches, comments per 1,000 views, saves, shares, and follow conversion.

## Quality Checklist

- Does the first 3 seconds contain a question or contradiction?
- Can the viewer explain the video in one sentence?
- Is there one main character and one main decision?
- Are facts, mainstream interpretations, and dramatized lines separated?
- Are character clothing and face consistent across shots?
- Does the ending invite comments without becoming clickbait?
