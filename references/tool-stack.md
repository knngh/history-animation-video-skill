# Tool Stack

Use this reference when recommending a production stack. Tool pricing, quotas, and availability change quickly. Verify official sources before giving current paid-plan advice.

## Selection Rule

Recommend by bottleneck, not by popularity:

- If scripts are weak, improve research and script prompts before paying for video.
- If visuals are inconsistent, pay for or standardize image generation first.
- If motion is unstable, use fewer shots and simpler movements before changing tools.
- If editing is slow, build a fixed editing template before hiring or subscribing.

## Budget Profiles

Free:

- Writing/research: free large-model tiers and public sources.
- Image: free credits from mainstream image tools or local Stable Diffusion.
- Video: rotate free credits from mainstream video tools.
- Voice/editing: CapCut/剪映 free features.
- Rule: produce 20 samples before paying.

Low:

- Pay for one main video generation tool only after sample validation.
- Pay for one editing/subtitle tool only if it saves repeated labor.
- Keep image, music, and title generation on free or low tiers when possible.
- Rule: one paid creative bottleneck at a time.

Pro:

- Use one reliable text model for scripts and fact-checking.
- Use a stable image workflow for character consistency.
- Use one premium video model for hero shots, cheaper tools for simple motion.
- Use professional voice if retention data proves narration matters.

Team:

- Standardize character sheets, prompt variables, file naming, and review checklist.
- Assign roles: research, script, visual, edit, publish, metrics.
- Build a shared prompt and asset library.

## Domestic-Oriented Stack

Use for Douyin, Xiaohongshu, Bilibili, and Chinese production:

- Script: DeepSeek, Kimi, 豆包, ChatGPT.
- Image: 即梦, 通义万相, Liblib, Stable Diffusion.
- Video: 可灵, 即梦, 海螺, plus other current mainstream tools after verification.
- Voice: 剪映配音, 火山/讯飞类 TTS, ElevenLabs if voice quality justifies cost.
- Edit: 剪映专业版, CapCut, DaVinci Resolve for advanced grading.
- Cover: Canva, 稿定, 剪映封面, image model plus manual text.

## Overseas-Oriented Stack

Use for YouTube Shorts, TikTok, and English/bilingual production:

- Script: ChatGPT, Claude, Gemini, DeepSeek.
- Image: Midjourney, Leonardo, Stable Diffusion.
- Video: Runway, Luma, Pika, Kling/可灵 if available.
- Voice: ElevenLabs, Azure/Edge TTS, platform-native voice if suitable.
- Edit: CapCut, DaVinci Resolve, Premiere if already owned.
- Assets: YouTube Audio Library and other license-clear libraries.

## Buying Checklist

Before paying, answer:

- Which exact workflow step is slow or low quality?
- Did the tool improve completion speed or retention in a test batch?
- Can the same result be achieved by a better prompt or simpler shot?
- Does the plan allow commercial use for the target platform?
- Are generated outputs watermarked?
- Can the user cancel monthly without annual lock-in?

Do not recommend a paid plan purely because it is famous. Tie every subscription to a production bottleneck and a measurable metric.
